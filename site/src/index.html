Principais projetos de pesquisa

Os Principais projetos de pesquisa dos alunos de Engenharia de Energia Bruna Fontenele Ferreira Silveira e Lucas Losinskas

Objetivos Principais

  1. Projeto de uma Bancada didática para a simulação de uma turbina hidráulica do rotor tipo Francis
  2. Analisar a percepção térmica e o conforto térmico de usuários de uma sala de aula, a partir de medições in loco;
  3. Dessalinizador solar
  4. Criação de um aplicativo didático do para o ensino de cálculo numérico;
  5. Sistema de controle de turbinas hidráulica;
  6. Criação de aplicativos e sites para o gerenciamento pessoal.
